【Kotlin】クイズアプリを作成した話

導入目的

  • Firebase（FireStore）の練習
  • クイズのスコアランキングを作成するため

完成図

以下が、実際に作成したランキング画面です。

 初期画面で入力したユーザーネーム、クイズ回答後に出力された正解数を FireStore に保存し、それらを正解数の降順にソートし、そのうち、5 件を取得してランキングを作成しています。

（デザイン良くない、、、）

ソースコード解説（一部）

 Firebase のセットアップは省略します。すでに有益な記事がネットに多くあります。

データの保存

 初期画面で入力した値、回答後の正解数を Score クラスに格納し、インスタンス化します。この 2 つの値は、Intent で持ってきてます。

 そして、この score を FireStore に保存します。今回は、コレクションを user_scores としています。

データの取得

 ランキング画面のアクティビティに遷移した際、データを取得してくるように、onCreate 内で readFireStore を呼び出します。

 今回は、取得してきたデータを ListAdapter で表示しています。詳しくは、私の Github アカウントのソースコードをご覧ください。

以上、終わります。
