読んでよかった技術書～2020下半期～


 今回は、僕が2020年下半期に読んで良かった技術書を紹介します。

目次

  • Web を支える技術
  • Node.js 超入門

  • イラスト図解式サーバーの基本

Web を支える技術

リンク

 Web について学ぶなら、この本！みたいなところもあると思います。出版日が少々前ですが、読んでおくべき一冊だと思いました。2 週目も復習がてら読みたいと思っています。

Node.js 超入門

リンク

 僕はこの書籍をきっかけに Node.js を学びました。この書籍では、Node.js の基本的な書き方や Express を用いたアプリケーション開発方法にも触れています。

Node クックブック

リンク

 この本では、上記の書籍と同様に Node.js について基本的なことを学ぶことが出来ます。基本的な web サーバーの構築方法、 MongoDB, MySQL, CouchDB などのデータベースとのインターフェース、 WebScoket の仕組みなど体系的に学ぶことが出来ます。

アルゴリズムとデータ構造入門

リンク

 この本では、アルゴリズムについて体系的に学ぶことが出来ます。

 この技術書を読んで学べることを一部列挙します。

  • 待ち行列・スタック
  • 連結・循環・双方向などのリスト
  • 木構造（二分探索木・B 木・AVL 木など）
  • 探索法（線形探索法・二分探索法）
  • ハッシュ法（チェイン法・オープンアドレス法）
  • クイック・バブル・挿入などのソート

 上記にあげたような基本的なアルゴリズムを Java で実装し学ぶことが出来ました。

マスタリング TCP/IP 入門

リンク

 TCP/IP まわりについて網羅的に学ぶことが出来ます。図表が多く、比喩も用いられており、とても分かりやすい内容でした。2 週目も復習がてら読みたいと思っています。

イラスト図解式サーバーの基本

リンク

 この本では、web サーバー、メールサーバー、DNS サーバーなどの各種サーバーの仕組みやセキュリティなどについて学ぶことが出来ます。マスタリング TCP/IP 入門よりも内容が浅いですが、分かりやすい内容になっていると思います。

イラスト図解式クラウドの基本

リンク

 クラウドの勉強の第一冊目として、この本を読みました。クラウドについて何も知らない状態でしたが、イラストが分かりやすいなどの工夫点があり、非常に読みやすいものでした。

AWS 仕組みと技術がしっかり分かる教科書

リンク

  Awazon Web Service で提供されているサービスを網羅的に学ぶことが出来ます。とりわけ、Amazon EC2, Amazon S3, Amazon VPC, Amazon RDS といった有名なサービスについて比較的詳しく解説されています。
