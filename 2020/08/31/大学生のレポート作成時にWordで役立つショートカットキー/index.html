大学生のレポート作成時にWordで役立つショートカットキー

はじめに

 俺的大学生がレポートを作成する際に Word で役立つショートカットまとめです。これを見て大学生の読者はレポート作成効率を爆上げしましょう！

Ctrl + A

 文章全選択　今押して試してみよう

Ctrl + C

 選択部分のコピー

Ctrl + V

 コピーしたものをペースト（貼り付け）

Ctrl + X

 カット（切り取り）
 Ctrl + C、Ctrl + V、Ctrl + X 　をよく組み合わせて使いますね

Ctrl + Z

 一つ前の状態に戻る。

Ctrl + Y

 前の動作をやり直す
 Ctrl + Z とよく合わせて使いますね。誤って一つ前の状態へ戻ってしまっても、Ctrl + Y でやり直すことが出来ます。

Ctrl + S

 上書き保存する。

Ctrl + Shift + S

 名前を付けて保存する。

Home

 カーソルを行の先頭に移動させる。
 カーソルを矢印キー（↑ ↓ ← →）で移動させるのはもうやめよう。

End 　

 カーソルを行の末に移動させる。
 Home と同様に便利なショートカットですね。

Ctrl + B

 選択部分を太字にする。

Ctrl + U

 選択部分に下線を引く

Shift + ↑ ↓ ← →

 選択部分の変更
 選択部分を矢印キーに応じて変更できます。

Ctrl + Shift + <

 選択部分のフォントの大きさを一段階上げる。

Ctrl + Shift + >

 選択部分のフォントの大きさを一段階下げる。

Ctrl + Shift + F

 フォント変更ウィンドウを出す。

最後に

 私はこれらのショートカットキーを覚えたら、レポートの作成効率が飛躍的に上がりました。みなさんも徐々にこれらのショートカットを使いながら覚え、レポート作成の効率を上げてみましょう。将来、Word で文書を作成する際にもきっと役立つショートカットばかりだと思います。
