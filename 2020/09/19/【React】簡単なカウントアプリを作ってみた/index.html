機能・ディレクトリ構造・完成イメージ

 このカウントアプリの機能は以下の通りです。

  • ＋ボタンを押すと、カウントが一つ増える
  • －ボタンを押すと、カウントが一つ減る
  • リセットボタンを押すと、カウントが０になる

 このアプリのディレクトリ構造は以下になります。

メソッドについて

 各ボタンに登録されているメソッドについて説明します。
handleCntUp
 setStateによって state の currentCnt の値を１だけ増やします。

handleCntDown
 setStateにって state の currentCnt の値を１だけ減らします。

handleCntReset
 setStateにって state の currentCnt の値を０にします。

 これらのメソッドはアロー関数で記述しています。メソッド記法で書くと、bind する必要があります。
