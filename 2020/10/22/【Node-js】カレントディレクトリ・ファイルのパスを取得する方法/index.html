process.cwd() と __dirname の違い

 上記のように、 process.cwd() や __dirname を使用すると、パスを取得できることが確認できました。では、これらの違いは何でしょうか。

  • process.cwd()は、現在のワーキングディレクトリのパスを取得するものです。このメソッドを実行した場所の絶対パスを取得できます。

  • __dirname は、実行した JavaScript ファイルがあるディレクトリの絶対パスを取得するものです。
