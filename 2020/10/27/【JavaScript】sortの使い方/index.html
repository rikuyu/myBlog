sort メソッドとは？

 sort メソッドは、配列やオブジェクトを並び替えることができるメソッドです。引数として、比較関数を受け取る場合が多いです。データ操作など様残な場面で利用されるメソッドです。

 この sort メソッドは、元の配列（または、オブジェクト）を並び替え後の配列（または、オブジェクト）で置き換えるかたちで実行されるため、破壊的メソッドといわれることもあります。元の配列を残しておきたい場合は、コピーを作成する必要があります。

 次の項目では、実際に sort メソッドの使い方をまとめていきたいと思います。

  • 戻り値が -1だった場合、a を b の前に移動させます。
    （1(=a)と 7(=b)を比較し、後者のほうが大きいため、-1 が返される、1, 7 の順に整列される）

  • 戻り値が、0だった場合、要素の移動は行われません。
